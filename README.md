# Проект Sneakers

Перед вами простая реализация интернет-магазина кроссовок на React. Макет верстки любезно предоставлен [Archakov Blog](https://www.youtube.com/c/ArchakovBlog), его можно посмотреть [тут](https://www.figma.com/file/fw0toTyXMwM1y4WIe0YFrJ/React-Sneakers?node-id=0%3A1).

В настоящее время проект находится на стадии разработки. Необходимо сделать:
- Адаптивную верстку;
- Написать логику.
